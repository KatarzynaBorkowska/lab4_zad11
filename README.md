Napisać program, który wyznacza:​

wektor sum liczb na odpowiadających sobie pozycjach (sumę wektorów przepisać do nowego wektora (tablicy) przekazywanej do funkcji liczącej sumę)​

oraz iloczyn skalarny (liczony jest dla dwóch wektorów {a1,a2,...,an} {b1,b2,...,bn} w następujący sposób: ​
il_sk = a1*b1 + a2*b2 + ... + an*bn)​

dwóch wektorów o jednakowej długości i dowolnych wartościach. Sumowanie, wyświetlanie sumy, liczenie iloczynu skalarnego,
wyświetlanie iloczynu skalarnego mają być realizowane w 4-rech oddzielnych funkcjach zaimplementowanych w jednym module podrzędnym, 
który powinien zostać poprawnie załączony do modułu nadrzędnego z funkcją main().
